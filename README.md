# Adventure Zones

Force players into Adventure mode when exploring structures for an extra challenge.

When a player moves close to a chunk containing a generated structure with a chest or a spawner, they are sent into [Adventure mode](https://minecraft.wiki/w/Adventure). In this mode, the player cannot directly place or break blocks, in order to avoid spoiling the structure. When they move outside again, they are returned to Survival mode.

The config allows you to specify which structures and blocks should create these "zones", as well as how much area they cover. By default, peaceful structures (villages, ruined portals, etc.) are blacklisted. To include modded structures, you need to add them to the config's whitelist manually, or make the whitelist empty to include all (non-blacklisted) structures.

Also check out my other mod, [Adventure Mode Adjust](https://github.com/gimme/adventure-mode-adjust), which lets you customize which blocks can be placed and broken in Adventure mode, using regular expressions!
